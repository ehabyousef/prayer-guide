
# Project Title

This program shows us prayer times in different Arab cities


## Demo

https://silly-squirrel-b16c62.netlify.app


## Run Locally

Clone the project

```bash
  git clone https://github.com/ehabyousef/prayer-guide
```

Go to the project directory

```bash
  cd prayer-guide
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
